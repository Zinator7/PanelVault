# 📖 Inkdex

> Un lecteur de scans de comics dynamique avec profils, classements, streaks et badges.

---

## ✨ Fonctionnalités

- **Lecteur de scans** — importez vos dossiers d'images (JPG/PNG) et lisez vos comics directement dans le navigateur
- **Profils utilisateurs** — avatar, niveau, statistiques de lecture personnalisées
- **Système de streak** — connectez-vous chaque jour pour maintenir votre série de lecture
- **Badges & achievements** — débloquez des badges selon vos actions (uploads, lectures, streaks...)
- **Dashboard personnel** — retrouvez tous vos comics uploadés, votre historique et votre progression
- **Classement hebdomadaire** — comparez vos points avec les autres lecteurs

---

## 🖥️ Stack technique

| Côté | Technologie |
|------|-------------|
| Frontend | HTML / CSS / JavaScript (Vanilla) |
| Backend | LocalStorage (pas de base de données) | ou phpmyadmin
| Lecture | FileReader API (import local) | ou autre...

> Inkdex est un **viewer client-side uniquement** — aucun scan n'est stocké sur un serveur.

---

## 🚀 Lancer le projet

```bash
# Cloner le dépôt
git clone https://github.com/ton-pseudo/inkdex.git

# Ouvrir index.html dans ton navigateur
# (ou utiliser Live Server sur VS Code)
```

Aucune dépendance, aucune installation requise.
